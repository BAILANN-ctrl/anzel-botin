# Personal Portfolio

My personal portfolio website built with Next.js. It showcases my projects, technical skills, and experience as a full-stack web developer.

## 🚀 Live Demo

https://your-portfolio.vercel.app

## ✨ Features

- Responsive design
- About Me section
- Skills overview
- Project showcase
- Contact information

## 🛠️ Built With

- Next.js
- React
- Tailwind CSS
- TypeScript (or JavaScript)
- Vercel

## 📦 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/portfolio.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.
